賺回防鏽70%的工程時間與最高19倍的投資報酬率
*******************************************

不必鏽搏鬥，讓鏽成為保護層

AR透明防鏽劑以凍鏽鈍化(ARCT, Autonomos Rust Condensation Technology)技術，讓活性壞鏽縮合為緻密穩定的鈍化層。適用於 C5X 以下嚴苛腐蝕環境，協助工程維護減少除鏽、縮短工期、降低長期維運成本。無論鋼構防鏽、海邊防鏽、鍍鋅防鏽、不鏽鋼(白鐵)防鏽、溫泉區防鏽，任何金屬物件無論有無生鏽都可使用。
