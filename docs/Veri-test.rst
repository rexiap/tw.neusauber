金屬鏽蝕怎麼開始？
------------------

A：眼睛看到的鏽都只是一連串反應的結果。鏽蝕反應是金屬被水中的氫氧基(OH-)啟動而開始，鏽蝕反應啟動後，鏽蝕活性就會在新鮮金屬間傳遞，接收到鏽蝕活性的金屬就變成奈米等級的鏽，又奈米鏽有非常多微孔洞，因毛細現象吸收更多空氣中的水氣，解離出越多氫氧基來啟動更多新鮮金屬轉成奈米鏽，不斷累積後就變成我們可以看到的鏽蝕。因此水與鏽蝕活性是防鏽首先要處理的兩個因素。
